JSModeler Generator Documentation
JSM.GenerateRectangle
Description
Generates a rectangle. The center of the rectangle will be the origo.
Parameters
xSize number x size of the rectangle
ySize number y size of the rectangle
Example
var body = JSM.GenerateRectangle (1.0, 1.0);
JSM.GenerateCuboid
Description
Generates a cuboid. The center of the cuboid will be the origo.
Parameters
xSize number x size of the cuboid
ySize number y size of the cuboid
zSize number z size of the cuboid
Example
var body = JSM.GenerateCuboid (1.0, 1.0, 1.0);
JSM.GenerateSegmentedRectangle
Description
Generates a rectangle with segmented side. The center of the rectangle will be the origo.
Parameters
xSize number x size of the rectangle
ySize number y size of the rectangle
segmentation integer segmentation number
Example
var body = JSM.GenerateSegmentedRectangle (1, 1, 4);
JSM.GenerateSegmentedCuboid
Description

Parameters

ySize number y size of the cuboid
zSize number z size of the cuboid
segmentation integer segmentation number
Example
var body = JSM.GenerateSegmentedCuboid (1, 1, 1, 4);

JSM.GenerateSphere
Description
Generates a sphere. The center of the sphere will be the origo.
Parameters
radius number radius of the sphere
segmentation integer segmentation of the sphere
isCurved boolean generate sphere with smooth surface
Example
var body = JSM.GenerateSphere (1, 10, false);
JSM.GenerateTriangulatedSphere
Description
Generates a sphere from triangles. The center of the sphere will be the origo.
Parameters
radius number radius of the sphere
segmentation integer segmentation of the sphere
isCurved boolean generate sphere with smooth surface
Example
var body = JSM.GenerateTriangulatedSphere (1, 2, false);
JSM.GenerateCylinder
Description
Generates a cylinder. The center of the cylinder will be the origo.
Parameters
radius number radius of the cylinder
height number height of the cylinder
segmentation integer segmentation of the cylinder
withTopAndBottom boolean generate cylinder with top and bottom surface
isCurved boolean generate cylinder with smooth surface
Example
var body = JSM.GenerateCylinder (0.5, 1.0, 15, true, false);
JSM.GeneratePie
Description
Generates a pie. The center of the pie will be the origo.
Parameters
radius number radius of the pie
height number height of the pie
angle number angle of the pie
segmentation integer segmentation of the pie
withTopAndBottom boolean generate pie with top and bottom surface
isCurved boolean generate pie with smooth surface
Example
var body = JSM.GeneratePie (1.0, 0.2, 270 * JSM.DegRad, 25, true, false);
